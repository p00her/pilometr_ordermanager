# Настройка MAX бота

## 1. Создать бота

1. Зайти в приложение MAX (или web-версию)
2. Создать бота через интерфейс платформы: https://dev.max.ru/
3. Задать имя и username (например `id071305521406_1_bot`)
4. Сохранить **токен бота**

## 2. Настроить мини-приложение (WebApp)

В настройках бота указать URL мини-приложения:
```
https://orders.pilometr.ru/max-app
```

## 3. Установить токен в endpoint.php

В файле `endpoint.php` заменить пустой токен:
```php
define('MAX_BOT_TOKEN', 'полученный_токен_бота');
```

## 4. Распространить бота среди продавцов

- Продавец устанавливает MAX на телефон
- Находит бота по `@id071305521406_1_bot`
- Открывает мини-приложение бота
- Вводит свой email (логин от системы заказов)
- Нажимает «Подключить»

## Принцип работы

- Бот **не имеет серверной логики** — он используется только как контейнер
  для мини-приложения (подключение) и токена (отправка сообщений)
- Все уведомления отправляются через MAX API из `endpoint.php`
- Список `chat_id` хранится в файле `max_chats.json` на сервере
- Нет callback-кнопок — только оповещения
